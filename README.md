emacs-live-packs
================

My emacs-live packs setup to use with the most excellent [[https://github.com/overtone/emacs-live][emacs-live]].

# pack list

They still need some love but here they are:

- user-install-packages-pack  some installation repository setup + function to install list of packages
- user-pack                   not yet sorted out stuff
- user-blog-pack              my blog setup to rapidly post article from org-mode
- user-haskell-pack           haskell environment setup
- user-orgmode-pack           some org-mode specifities
- user-lisp-pack              list environment setup
- user-buffer-pack            some buffer manipulation functions
- user-git-pack               git specific setup
- user-mail-pack              email setup
- user-shell-pack             shell setup
- user-browser-pack           browser specific tryouts
- user-chat-pack              irc setup
- user-clojure-pack           clojure override from emacs-live's defaults + other stuff
- user-nrepl-pack             nrepl override setup
- user-clojurescript-pack     clojurescript setup
- user-caml-pack              some caml setup

# install

## clone

Clone the repository.

```sh
git clone git@github.com:ardumont/emacs-live-packs.git
```

## deploy

Launch the deployer:

```sh
/path/to/emacs-live-packs/deploy.sh
```

This will trigger the creation of 2 links in your home folder.

Example:

```sh
tony@dagobah(0,43,) 15:30:39 ~/repo/perso/emacs-live-packs (master) $ ll ~/.emacs*
lrwxrwxrwx 1 tony tony 32 dÃ©c.  18 03:28 /home/tony/.emacs.d -> /home/tony/repo/perso/emacs-live
lrwxrwxrwx 1 tony tony 53 avril 15 15:30 /home/tony/.emacs-live.el -> /home/tony/repo/perso/emacs-live-packs/.emacs-live.el
lrwxrwxrwx 1 tony tony 39 avril 15 12:38 /home/tony/.emacs-live-packs -> /home/tony/repo/perso/emacs-live-packs/
```

Now start emacs.

```sh
emacs --daemon
```
